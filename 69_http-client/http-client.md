#### V's `net.http` module provides `http.get()` for simple HTTP requests. The response body is returned as a string; split on newlines to process line by line.
___
##### Run Command:

`$ v run http-client.v`

##### Results:

`Response status: 200 OK`
`<!DOCTYPE html>`
`<html>`
`  <head>`
`    <meta charset="utf-8">`
`    <title>Go by Example</title>`
