#### V's `time` module provides `unix_time()` for seconds and `unix_time_milli()` for milliseconds since the Unix epoch. Nanosecond-precision epoch access is not available in the standard module.
___
##### Run Command:

`$ v run epoch.v`

##### Results:

`2009-11-17 20:34:58`
`1258490098`
`1258490098651`
`2009-11-17 20:34:58`
`2009-11-17 20:34:58`
